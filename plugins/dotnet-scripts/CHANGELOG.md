# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-03-08

### Added

- dotnet-scripts skill for writing and running single-file C# programs with .NET 10+ file-based apps
- dotnet-tools skill covering the full .NET tool lifecycle: create, pack, publish, and run via dnx
- script-to-tool-promoter agent for promoting scripts into distributable .NET tools
- script-migrator agent for converting Python or Bash scripts to C# file-based apps
- Reference docs for file-based app directives, dnx, user secrets, and common pitfalls
