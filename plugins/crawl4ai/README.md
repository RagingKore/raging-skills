# Crawl4AI

Complete toolkit for web crawling and data extraction using the Crawl4AI library.

## Overview

Crawl4AI is a skill plugin that provides comprehensive support for web crawling and
structured data extraction, from simple markdown extraction to complex multi-URL batch
processing and schema-driven data pipelines. It includes ready-to-use scripts, optimized
extraction patterns, and full SDK reference documentation.

## Skills

### Auto-Loaded

**crawl4ai**

Activates when you work on web scraping, data extraction, or crawling tasks. Guides setup
detection, extraction strategy selection, crawl configuration, and output formatting.
