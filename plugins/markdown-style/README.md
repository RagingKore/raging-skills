# Markdown Style

Markdown writing conventions for consistent formatting across all `.md` files.

## Overview

This plugin enforces a consistent set of markdown writing conventions covering line length, heading
hierarchy, list formatting, link style, code block language hints, table alignment, and common
anti-patterns to avoid. It activates before writing or editing any `.md` file, ensuring that prose,
structure, links, lists, code blocks, and tables follow the same rules across a project.

## Skills

### Auto-Loaded

**Markdown Style**

Activates before writing or editing any `.md` file, plan, doc, PR description, commit message,
report, or any markdown content. Provides conventions for prose, structure, links, lists, code
blocks, and tables.
