# Mermaid

Write correct Mermaid diagram syntax for all 23+ diagram types with expert-level precision.

## Overview

The mermaid plugin provides Claude Code with a complete syntax reference for all Mermaid diagram
types: flowcharts, sequence diagrams, class diagrams, state diagrams, ER diagrams, gantt charts,
pie charts, gitgraph, mindmaps, timelines, quadrant charts, sankey diagrams, and more. It covers
node shapes, edge styles, subgraphs, configuration, theming, layout engines, and common gotchas
so you can produce correct Mermaid markup the first time.
