# Proto Style

Protocol Buffer style conventions for edition 2023, based on Google AIPs.

## Overview

This plugin provides style conventions for writing Protocol Buffer definitions targeting edition
2023. It covers field naming, comment style, enumerations, message structure, pagination, timestamps,
durations, resource identifiers, and the migration path from proto3. The conventions draw from
Google's API Improvement Proposals (AIPs) and the official Protocol Buffers style guide.

## Skills

### Auto-Loaded

**Proto Style**

Activates when you write or edit `.proto` files, review protobuf definitions, design gRPC APIs, add
new messages or fields, or migrate from proto3 to edition 2023. Provides guidance on field naming,
presence rules, enumeration conventions, pagination patterns, and edition 2023 migration steps.
