## [1.0.0] - 2026-03-08

### Added

- DDD skill covering strategic patterns (bounded contexts, context mapping, subdomains) and tactical patterns
  (aggregates, entities, value objects, domain events)
- Implementation patterns: Decider/Functional, OOP, and Actor-based
- Event sourcing and CQRS architecture guidance with .NET and Python examples
- Reference guides for projections, event evolution, and actors
