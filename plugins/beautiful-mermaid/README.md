# Beautiful Mermaid

Render Mermaid diagrams as ASCII art or themed SVG files using beautiful-mermaid via bun.

## Overview

The beautiful-mermaid plugin renders Mermaid diagrams into ASCII/Unicode art for your terminal
or themed SVG files saved to disk. It handles environment bootstrapping, rendering execution via
`bun -e` commands, and provides 15 built-in color themes. Where the mermaid plugin writes correct
Mermaid syntax, this plugin actually renders the diagram.

## Skills

### Auto-Loaded

**beautiful-mermaid**

Activates when you ask to draw a diagram, create a flowchart, visualize architecture, render
mermaid, or generate ASCII art. Handles bun environment bootstrapping and rendering execution
for flowcharts, state diagrams, sequence diagrams, class diagrams, and ER diagrams.
