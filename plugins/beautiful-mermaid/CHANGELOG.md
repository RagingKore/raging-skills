## [1.0.0] - 2026-03-08

### Added

- Mermaid rendering skill with ASCII art and themed SVG output via bun
- Complete syntax reference for flowchart, state, sequence, class, and ER diagram types
- Self-bootstrapping environment setup with auto-installed bun package
- 15 built-in themes for SVG rendering
- Reference docs for execution patterns and real-world diagram examples
