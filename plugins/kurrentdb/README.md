# KurrentDB Plugin

Expert-level guidance for KurrentDB event-native database development.

## Overview

KurrentDB (formerly EventStore) is an event-native database built for event sourcing, event-driven architecture,
and real-time event streaming. This plugin provides guidance on the .NET KurrentDB.Client API, streams, projections,
subscriptions, connectors, indexes, server configuration, security, operations, and event-driven architecture patterns
like outbox, time-travel, and polyglot persistence.

## Skills

### Auto-Loaded

**KurrentDB**

Activates when you mention KurrentDB, EventStore, EventStoreDB, event sourcing database, event store, event streams,
or event-native database. Helps with writing event sourcing code, managing streams and subscriptions, configuring
projections and connectors, and operating KurrentDB clusters.
