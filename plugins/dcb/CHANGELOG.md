# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-08

### Added

- Dynamic Consistency Boundary skill with decision model pattern replacing traditional aggregates
- Tag-based event queries and conditional append operations
- Implementation patterns with C#/.NET examples and event store adapters for KurrentDB, Marten, PostgreSQL
- Migration guide from traditional event sourcing with anti-patterns and common mistakes guide
