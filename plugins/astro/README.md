# Astro Plugin

Comprehensive guidance for building and deploying static websites with the [Astro](https://astro.build/) framework. Covers SSG fundamentals, Content Collections, Markdown/MDX, partial hydration, islands architecture, Starlight documentation framework, and deployment to multiple platforms.

## Skills

| Skill     | Purpose                                                                                    |
|-----------|--------------------------------------------------------------------------------------------|
| **astro** | Full Astro framework guide — SSG, content collections, Starlight, Markdown/MDX, deployment |
